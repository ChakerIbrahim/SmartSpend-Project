Hello World Assignment

Objective:
This assignment helped me practice different ways of printing and concatenating strings in Python.

What I learned:

1. Printing Strings:
I learned how to print simple text using the print() function.

2. Using Variables:
I used variables to store values like names and numbers, and then printed them.

3. Concatenation Methods:
- Using comma: separates values and automatically adds space
- Using +: joins strings together but requires same data type

4. Type Error:
When using + with a string and an integer, Python gives an error.
I learned that I need to convert integers to string using str().

5. String Formatting:
- .format() method: older but useful
- f-strings: modern and easier way to insert variables into strings

Conclusion:
This assignment helped me understand basic Python syntax, string handling, and the importance of data types.
