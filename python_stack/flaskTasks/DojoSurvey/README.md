# Champion Survey

## Description
Champion Survey is a simple Flask web application that allows users to fill out a survey form and submit their information.

The application demonstrates:
- Flask routing
- HTML forms
- POST requests
- Rendering templates with Jinja
- Passing data from client to server

---

## Features
- User enters:
  - Name
  - Dojo Location
  - Favorite Language
  - Comment
- Submitted data appears on a result page
- Styled using Tailwind CSS

---

## Technologies Used
- Python
- Flask
- HTML
- Tailwind CSS
