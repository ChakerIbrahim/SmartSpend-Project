# Great Number Game

A simple Django game where the user tries to guess a random number between 1 and 100.

---

## Features

- Random number generation
- Too High / Too Low messages
- Correct answer message
- Session usage
- Attempt counter
- Maximum 5 attempts
- Lose message
- Play Again button
- Simple CSS styling

---

## Technologies Used

- Python
- Django
- HTML
- CSS
